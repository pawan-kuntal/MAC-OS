# 🍎 macOS-Inspired Interactive Portfolio

A fully interactive macOS-style desktop interface built using **React + Vite**.  
This project replicates the behavior of a real operating system UI inside the browser — including dynamic windows, dock interactions, and app-style navigation.

> Not a static portfolio. It behaves like an application.

---

## 📂 GitHub Repository
https://github.com/pawan-kuntal/MAC-OS

---

## 🛠 Tech Stack

- React (Functional Components + Hooks)
- Vite (Lightning-fast build tool)
- JavaScript (ES6+)
- SCSS / CSS
- Component-based Architecture

---

## ✨ Features

- 🖥 macOS-style Dock with app open/close behavior
- 🗂 Dynamic window rendering system
- 🧠 Centralized state-driven UI control
- 🖱 Interactive app windows
- 📄 Resume Viewer
- 💻 Terminal-style interface
- 🎵 Spotify-inspired UI
- 🌐 GitHub profile window
- 📅 Live Date & Time display
- ♻ Reusable window components

---

## 🏗 Architecture Overview

This project is built around:

- Reusable Window Component Pattern
- Conditional Rendering Logic
- State-driven UI management
- Component composition structure
- Clean separation of concerns

The window system dynamically renders applications based on state, simulating real desktop behavior.

---

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/pawan-kuntal/MAC-OS.git
```

Navigate to the project folder:

```bash
cd MAC-OS
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
